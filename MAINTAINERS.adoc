== Review policy

* At least one maintainer review before merge to `main`.
* Lean proof changes must `lake build` cleanly (toolchain pinned in
  `lean-toolchain`); Isabelle theory changes must pass `just isabelle-build`
  (the `Tropical_Semirings` session). Proofs are load-bearing — a red build
  blocks merge, and `just check-sorry` must report zero placeholders.
